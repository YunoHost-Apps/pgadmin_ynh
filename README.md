Pgadmin for yunohost
========================

*https://www.pgadmin.org

Install
-------

From command line:

`sudo yunohost app install -l Pgadmin https://github.com/YunoHost-Apps/pgadmin_ynh`

Upgrade
-------

From command line:

`sudo yunohost app upgrade -u https://github.com/YunoHost-Apps/pgadmin_ynh`
